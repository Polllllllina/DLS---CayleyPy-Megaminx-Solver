# DLS---CayleyPy-Megaminx-Solver

Поиск кратчайших путей на графах Кэли для головоломки Megaminx с использованием нейронных сетей и beam search.

## Описание задачи

Нахождение последовательности ходов для сборки Megaminx — головоломки с 120 элементами и 24 возможными ходами (12 граней с прямым и обратным вращением). Пространство состояний имеет размер ~10^68, полный перебор невозможен.

Задача сводится к поиску пути на графе Кэли (Cayley graph). Аналогичные задачи возникают в квантовых вычислениях, биоинформатике, теории групп и оптимизации сетей.

## Подход

Решение основано на трёхшаговом подходе:

1. Генерация обучающих данных случайными блужданиями (random walks) из собранного состояния в режиме non-backtracking
2. Обучение нейронной сети (Residual MLP) предсказывать расстояние до цели
3. Решение beam search с использованием предсказаний сети как эвристики

## Конфигурация

| Параметр | Значение |
|----------|----------|
| Архитектура | ResMLP 256→128×1 |
| Параметры | 3.75M |
| Обучение | Stage0: 1000 эпох Adam (lr=4e-3) + Stage1: 250 эпох AdamW (lr=4e-4) |
| Функция потерь | PinballLoss (tau=0.9) |
| EMA | decay=0.999 |
| Gradient clipping | max_norm=10.0 |
| Beam width | 1024 |
| History depth | 10 |
| Max steps | 1000 |

## Выводы

1. Многостадийное обучение (Adam -> AdamW с weight decay) улучшает сходимость
2. Увеличение beam width уменьшает длину путей
3. History depth предотвращает циклы в beam search
4. Pinball loss работает лучше MSE для задач планирования путей
5. EMA усреднение весов стабилизирует обучение
